Detailed Description

template<typename T>
struct cgv::type::info::type_name< T >

traits class with a static function get_name() of type const char* that returns the type name of the template argument T.

The standard types have the following textual names:

  • "bool"
  • "int[8|16|32|64]"
  • "uint[8|16|32|64]"
  • "flt[32|64]"
  • "wchar"
  • "string"
  • "wstring"

all other types are derived from the RTTI with the extract_type_name function applied to the type_info structure returned by typeid.
